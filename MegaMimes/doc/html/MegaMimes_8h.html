◆ getMegaMimeExtensions()

char** getMegaMimeExtensions ( const char *  pMimeType)

Get The Extensions associated with Mime Types.

The function gets the file extensions for the mimetype. The mimetype should be in the form type/sub-type, otherwise it is invalid. An optional version can be added to the string. Any other trailing details are ignored. * character can be used to match everything. Eg video/ returns all video file extensions and * returns all file extensions

Parameters
pMimeTypeThe mimetype whose extension is to be determined
Returns
An array of extension names. The array is terminated by a NULL. The function returns NULL if no extensions were found for the mime types given.

◆ getMegaMimeType()

char* getMegaMimeType ( const char *  pFileName)

Get The Mime Type Of A File Or A File Extension.

The function gets the mime type of the file specified by the path. The function does not check if the file exists or not. It just uses the file's basename including the extension to lookup the mime type. The filename can also be an extension only, which is preceeded by a ".". Example (.mp3, mega.mp3, /usr/lib/mega.mp3 ) will all return the same thing.

Parameters
pFileNameThe name of the file whose mime type should be determined
Returns
the mimetype of the file or NULL if no mimetype is known for the file. The string returned should be freed

const char* getMegaTextFileEncoding ( const char *  path)

Gets the encoding for a text file.

Precondition
The file is assumed to be a text file. Use isTextFile() to check the file before calling this function.

The function gets the encoding of the file. The string can be UTF-8, UTF-8, UTF-16LE, UTF-16 UTF-32, UTF-32BE, UTF-32LE. The function does not check if the file is binary file or a text file. It is assumend that you are really sure that it is a binary file. You can use isTextFile() to first check if it is a text file, before using the function.

Parameters
pathThe path name for the file
Returns
the file encoding or empty string if the file does not exist or cannot be read.
